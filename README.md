# core

> Core functions for setting results, logging, registering secrets and exporting variables across actions.

## Usage

### Import the package

```dart
export 'package:actions_toolkit_dart/core.dart' as core;
```

#### Logging

```dart
export 'package:actions_toolkit_dart/core.dart' as core;

// Do stuff
core.info('Output to the actions build log');
```
